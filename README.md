# Discord.jar

A Robust API and Websocket wrapper for Discord

---

### Setting Up
```java
public class Main {

    private final String TOKEN = "YOUR_SECRET_BOT_TOKEN";
    
    public static void main(String[] args) {
        
        DJAR djar = new DJARManager(TOKEN).buildClient();
    }

}
```
